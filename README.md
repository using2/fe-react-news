## 요구사항 정리

### 전체 요구사항

- [x] 콘텐츠 웹 화면 정중앙

### 기본상단 영역

- [x] 왼쪽 뉴스스탠드 로고
- [x] 오른쪽 시스템 날짜 표시

### 최신 뉴스 자동롤링 영역

- [x] 왼쪽 바와 오른쪽 바는 각각 다른 최신 뉴스의 헤드라인 5개
- [x] 모든 바 5초마다 자동으로 무한 롤링
- [x] 좌우 영역의 시간차를 1초
- [ ] 롤링 시, 바 영역 안에서 제목이 위로 넘어가는 애니메이션
- [ ] 각 영역에 마우스를 hover하면, 무한 롤링을 일시정지
- [ ] 각 영역에 마우스를 hover하면, 헤드라인에 밑줄
- [ ] 마우스가 해당 영역에서 벗어나면 다시 1초 간격의 차이를 두고 두 영역의 뉴스가 다시 롤링
- [ ] 링크에 hover할 때는 항상 마우스커서는 손 모양

### 기사 확인하는 메인 영역

- [ ] [전체 언론사]탭의 [그리드 보기]를 기본 상태
- [ ] [내가 구독한 언론사]탭의 타이틀 옆에는 badge 컴포넌트로 내가 구독한 언론사의 숫자를 표시

### 1. 전체 언론사 : 그리드 보기

- [ ] 가로 930px, 세로 388px의 영역에 6\*4의 테이블
- [ ] 각 테이블의 셀에는 언론사 브랜드마크가 중앙에 배치
- [ ] 언론사 브랜드 마크의 순서는 페이지가 새로고침 될 때마다 랜덤으로 배치
- [ ] 그리드의 좌우에는 화살표를 배치하여 페이지를 넘길 수 있도록 구현
- [ ] 가장 첫 페이지와 가장 끝 페이지에서 Disabled 되는 화살표는 아예 표시하지 않음
- [ ] 언론사는 최대 4페이지 까지만 표시
- [ ] 각 언론사 브랜드마크가 있는 셀에 마우스를 올리면, [구독하기]버튼 표시
- [ ] 이미 구독하고 있는 언론사의 경우, [해지하기]버튼이 표시

### 2. 전체 언론사 : 리스트 보기

- [ ] 각 분야 카테고리에서 언론사의 기사를 표시
- [ ] 종합/경제, 방송/통신, IT, 영자지, 스포츠/연예, 매거진/전문지, 지역 카테테고리는 기사 영역 상단에 가로로 긴 탭 형태로 배치

### 3. 내가 구독한 언론사 : 리스트 보기

- [ ] ‘내가 구독한 언론사의 리스트 보기’ 화면은 ‘전체 언론사의 리스트 보기' 화면과 동일하되, 카테고리들이 나열되어있던 가로 탭에 언론사의 이름과 > 화살표 아이콘 추가로 표시
- [ ] 언론사가 많아서 탭의 가로 영역을 넘어가는 경우, 드래그를 통해 가로로 스크롤
- [ ] 언론사는 구독한 순서대로 배치
- [ ] 전체 언론사 탭 상태에서는 그리드보기가 기본인 것처럼, 내가 구독한 언론사 탭 상태에서는 리스트 보기로 기본 설정
- [ ] [구독해지]버튼을 누르면 “{언론사 이름}을(를) 구독해지하시겠습니까?”라는 알럿 구현 (예, 해지합니다, 아니오)
- [ ] 각 버튼에도 마우스를 올리면 텍스트에 밑줄 표시
- [ ] ‘예, 해지합니다'를 선택하면 즉시 구독이 해지되고, 목록의 다음 순서 언론사가 바로 표시

### 4. 내가 구독한 언론사 : 그리드 보기

- [ ] 내가 구독한 언론사의 그리드 보기 화면은 ‘전체 언론사의 그리드 보기’ 화면과 동일
- [ ] 사용자가 구독을 누른 언론사의 브랜드마크만 보이고, 나머지 칸은 비어있게 표시
- [ ] 구독중인 언론사의 브랜드마크 셀에 마우스를 올리면 [구독해지]버튼이 표시
- [ ] 리스트보기 상태에서 구독해지를 하는 것과 동일하게, 알럿 뜨도록 표시
- [ ] 구독이 해지되는 즉시 그리드에서 해당 언론사의 브랜드마크가 사라지도록 구현
